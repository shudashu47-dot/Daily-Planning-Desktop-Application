# 📋 每日计划

一个简洁、美观、零配置的每日计划制定桌面程序。

基于 **PySide6** 构建，支持任务管理、番茄钟、统计分析和数据导出，所有数据本地存储，无需联网，开箱即用。

---

## ✨ 功能特性

| 功能 | 说明 |
|------|------|
| ✅ 任务管理 | 添加、编辑、删除、完成标记，支持优先级和分类标签 |
| 🏷 优先级 | 高 / 中 / 低 三档，左侧彩色竖条直观区分 |
| 📂 分类标签 | 工作、学习、生活、其他，彩色标签一目了然 |
| 🍅 番茄钟 | 15/25/45/60 分钟可选，专注倒计时，完成提醒 |
| 📊 统计分析 | 近 7 天完成率趋势、任务总量、番茄数统计 |
| 📤 数据导出 | 支持 Markdown、CSV、JSON 三种格式导出 |
| 📝 每日备注 | 随时记录当天的想法、反思或重要事项 |
| 💾 本地存储 | 所有数据以 JSON 格式保存在本地，透明可备份 |

## 🚀 快速开始

### 方式一：直接运行 EXE（推荐，无需安装 Python）

1. 下载 `dist/` 文件夹
2. 双击 `每日计划.exe`
3. 数据自动保存在 `dist/data/` 目录下

### 方式二：从源码运行

```bash
# 1. 克隆或下载本项目
cd daily-planner

# 2. 安装依赖
pip install -r requirements.txt

# 3. 运行
python main.py
```

### 方式三：Windows 双击启动（源码方式）

直接双击目录下的 `每日计划.vbs`，无黑窗静默启动。

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + N` | 聚焦到添加任务输入框 |
| `← / →` | 切换上一天 / 下一天 |
| `Ctrl + T` | 回到今天 |
| `Ctrl + S` | 打开统计窗口 |
| `Enter` | 确认添加任务 / 确认编辑 |
| `Esc` | 取消编辑弹窗 |

## 💾 数据与备份

所有数据以 **纯 JSON** 格式存储在 `data/` 目录下，每天一个文件：

```json
{
  "date": "2026-05-01",
  "tasks": [
    {
      "id": "xxx",
      "content": "完成任务",
      "priority": "high",
      "category": "工作",
      "completed": true,
      "created_at": "...",
      "completed_at": "...",
      "tomato_count": 2
    }
  ],
  "daily_note": ""
}
```

**备份方法**：直接复制 `data/` 文件夹即可。

**迁移方法**：将旧电脑的 `data/` 文件夹复制到新电脑的相同位置，所有历史记录完整保留。

## 🛠 技术栈

- **GUI 框架**：PySide6 (Qt6)
- **语言**：Python 3.9+
- **数据存储**：JSON 文件
- **打包工具**：PyInstaller

---

## 📦 重新打包

如需自行打包成独立 EXE：

```bash
pip install pyinstaller
python -m PyInstaller --onefile --windowed --name "每日计划" main.py
```

打包完成后，将 `data/` 目录复制到 `dist/` 文件夹旁即可。
